# php-hello-world
A simple hello-world for composer

Installation
------------

Install Docker and Docker Compose
``` bash
sudo apt install docker
```
``` bash
sudo apt install docker.io
```
``` bash
install docker-compose
```

Install Composer and use Docker Compose
-------------

Install Composer
``` bash
composer Install
```

Run docker-compose.yml
``` bash
docker-compose up -d
```
